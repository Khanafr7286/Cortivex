# Cortivex Documentation

This folder contains the reference documentation for Cortivex, an AI agent pipeline orchestration system for Claude Code.

## Contents

- **[Getting Started](getting-started.md)**: Installation, first pipeline run, custom pipeline creation, and understanding output.
- **[Skill Reference](skill-reference.md)**: All 15 skills with categories, capabilities, activation behavior, and skill file structure.
- **[API Reference](api-reference.md)**: MCP server tools and HTTP REST endpoints with parameters and usage examples.
- **[Templates](templates.md)**: All 15 pipeline templates with costs, durations, customization, and YAML format reference.
